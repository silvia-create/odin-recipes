# Odin Recipes
A simple recipe website built as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## What I practiced
- HTML structure and anchor elements
- Relative links and URLs
- Cloning a repository from GitHub
- Writing a README and making commits

## What I learned
Setting up SSH keys and connecting to GitHub via port 443 when port 22 is blocked.
